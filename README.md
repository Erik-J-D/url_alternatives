## Confusable Domain Name Generator

This script generates potential candidates for lookalike domain names, replacing regular english characters with simalar unicode ones.

## Usage

```
$ ./url_generator.py asd.com
⍺ｓⅾ.com

⍺ｓ𝓭.com
⍺ｓ𝔡.com
⍺ｓ𝕕.com
⍺ｓ𝖉.com
...
```
